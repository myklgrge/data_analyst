🎧 Tamil Music’s Most Popular Voices: A Data Analytics Report (2020–2024)

A professional analysis of streaming trends, gender representation, and song popularity in Tamil music.
By Micheal George

Key Findings:
  • 🔵 Male-fronted tracks consistently topped the charts every year.
  • 🟣 Duet numbers were a strong second, pairing male and female voices.
  • 🟡 All-female tracks were rare in the top 50.

With Python, SQL, and Tableau, raw streaming data was transformed into interactive dashboards and clear insights. Explore the visuals and analysis below.

🟨 1. Data Collection

For this project, I manually curated the top 50 Tamil songs for each year between 2020 and 2024. The data was sourced from publicly available streaming charts and popularity lists (e.g., YouTube and Spotify rankings).


    However, the raw data wasn’t clean — it came with several unnecessary columns like:

  • Loudness

  • YouTube upload time

  • View counts

  • Thumbnail URLs

  • Movie titles

  • Language metadata
    …and more.

To make the dataset suitable for analysis, I performed an initial cleaning using Excel, where I:

  • Removed irrelevant columns

  • Renamed headers for consistency across years

  • Retained only the essential data for this project

Each year’s cleaned dataset included:

  • 🎵 Track Name

  • 🎤 Artist Name(s)

  • ⭐ Popularity Score (standardized between 0 and 100)

  • 📅 Year of Appearance

Note: These songs are not necessarily from albums released in that specific year, but represent the top-streamed Tamil tracks during that year.


🟨 2. Data Cleaning in Python

Using Python (Pandas), I processed and cleaned the data as follows:

  • Standardized column names across all years

  • Split multiple artists from the same song into separate rows using .explode()

  • Created a mapping of individual artists to their gender (male, female, or unknown)

  • Labeled each song as Male, Female, or Duet based on the combination of artists

         Will share the .py file and sql script through github repo


🟨 3. SQL Data Analysis

After cleaning, the dataset was loaded into MySQL for further analysis.
I joined all five years into one master table with a year column, and used SQL to answer key questions like:

    Examples:

  • How many songs each year were Male, Female, or Duet?

  • What's the average popularity by gender type?

  • What gender type dominates the most popular tracks?
